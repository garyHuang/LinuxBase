# chmod权限命令

## chmod 授权
```
用数字形式表示 r=4 w=2 x=1 rwx=7 rx=6 rw=6
u表示当前用户
g表示当前组
o表示其他用户
a表示全部
# 给脚本设置所有用户可读可写可执行权限
chmod a=rwx 1.sh
# 给脚本设置当前用户可读可写可执行权限
chmod u=rwx 1.sh
# 给当前组和其他用户设置可读可写可执行权限
chmod ao=rx 1.sh

# 给文件去掉可执行的权限
chmod a-x 1.sh
# 给文件加上可执行权限
chmod a+x 1.sh 

# 多个权限可以都逗号隔开, 给当前用户可可写可读可执行权限，同组用户可读权限，其他用户去掉可执行权限
chmod u=rwx,g=r,o-x 1.sh 
root 账号可以读取和修改所有文件
```
